类型定义

​ Groovy 是有 两种类型定义 方式，分别是 强定义 和 弱定义 , 如果其他类或方法要是使用定义的变量，那么类型就要使用 强类型 ，如果仅仅是使用于当前的方法中的话可以使用 弱类型 定义，因为 强类型 定义的不能动态转换成其他类型，而 弱类型 可以。

强类型 就和 java 定义类型是一样的，使用 int char 等，强制指定数据的类型。

弱类型 是使用 def 关键字进行指定，编译器会在 解析变量值 时进行推断具体是什么数据类型，支持动态类型转换。
